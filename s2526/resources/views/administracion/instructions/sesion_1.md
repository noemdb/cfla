# Sesión 1: Introducción al sistema y navegación general del módulo de cobranzas

## Objetivo
Familiarizar al operador con el entorno del sistema SAEFL, el acceso seguro y la navegación básica dentro del módulo de cobranzas.

## Ejercicios Desarrollados

### Ejercicio 1: Ingreso al Sistema (Login)
**Objetivo:** Acceder correctamente al sistema con credenciales de usuario.
**Pasos:**
1. Abra el navegador web y diríjase a la URL del sistema.
2. En la pantalla de inicio de sesión, ingrese su usuario y contraseña asignados.
3. Haga clic en el botón "Ingresar".
4. Verifique que el sistema muestre el panel principal (Dashboard) con su nombre de usuario en la esquina superior derecha.
**Resultado Esperado:** Acceso exitoso al Dashboard principal.

### Ejercicio 2: Exploración del Menú de Navegación
**Objetivo:** Identificar las secciones principales del módulo de cobranzas.
**Pasos:**
1. Ubique la barra lateral izquierda (o menú superior).
2. Haga clic en la sección "Administración".
3. Despliegue el submenú "Cobranzas" o "Pagos".
4. Identifique las opciones referidas a: "Asistente - Registro de Pago", "Reportes", "Asistente D. Individual", etc.

### Ejercicio 3: Cierre de Sesión Seguro
**Objetivo:** Finalizar la sesión de trabajo de manera segura.
**Pasos:**
1. Ubique su nombre de usuario o icono de perfil en la esquina superior derecha.
2. Haga clic para desplegar el menú de usuario.
3. Seleccione la opción "Cerrar Sesión" o "Salir".
4. Confirme que el sistema redirige a la pantalla de inicio de sesión.
**Resultado Esperado:** El sistema cierra la sesión y vuelve a la pantalla de login.

**Meta:** Familiarizarse con la información visual inmediata del sistema.
